# ObreshkovLibrary

## Описание
ObreshkovLibrary е уеб базирана библиотечна система, предназначена за училищна библиотека. Проектът улеснява управлението на книги, категории, читатели и заемания, като предоставя отделни възможности за администратор, ученик и гост.

## Цел на проекта
Целта на проекта е да се създаде удобна и функционална система за организация и управление на библиотечен фонд в училищна среда. Чрез нея се подобрява работата по добавяне, търсене, заемане и проследяване на книги.

## Използвани технологии
- C#
- ASP.NET Core MVC
- Razor Views
- Entity Framework Core
- SQL Server
- HTML
- CSS
- JavaScript
- Bootstrap

## Основни функции
- вход в системата с потребителски роли;
- административно управление на книги;
- управление на категории;
- управление на читатели;
- заемане и връщане на книги;
- ученически профил;
- любими книги;
- известия;
- публичен каталог за гости.

## Инструкции за стартиране
1. Отворете проекта във Visual Studio.
2. Проверете връзката към базата данни в конфигурационните файлове.
3. Изпълнете миграциите към базата данни, ако е необходимо.
4. Стартирайте проекта чрез IIS Express или стандартния профил за стартиране.
5. Отворете приложението в браузър.

## Кратко описание на употребата
Администраторът използва системата за управление на библиотечните данни,
книгите, категориите, читателите и заеманията. Ученикът може да разглежда книги,
да следи профила си, да добавя любими заглавия и да получава известия.
Гостът може да разглежда публичната част на каталога без вход в системата.
